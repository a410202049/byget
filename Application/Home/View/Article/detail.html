Web前端开发：初识HTML
柴迪一 2016年1月10日 15:12 Python Java
我是14年毕业的，大学的时候学的也是编程。由于大学根本没有用心学，所以刚毕业时候很多面试都没有通过。当时很慌乱，毕业了一个月了都还没有找到工作。所以也就没有执着于编程方面，就找了份通讯方面的工作。 但是专业方面不对口，又对这方面没有兴趣，所以有了转行的想法。于是就报了麦子学院的《Python Web 开发》的课程，通过在麦子四个月的学习，还有李希老师的帮助，现在已经成功找到了python方面的工作。在此跟大家分享一下我的就业经历。 首先跟大家说一下我对第一份工作的反思。很多大学生都有一个通病，就是“混”！我当然也是这样，四年一转眼就混过去了。到了找工作的时候就完全傻眼了，只能匆匆找一份工作应付了事，现在想来当时真是太急躁了。其实编程方面的应届生是很容易找到工作的。如果当时可以踏实的学一期课程，现在也不用顶着压力去转行了。所以在这里提醒一下还是在校生的同学们，好好珍惜在麦子学习的机会，这会使你的应届生面试变得非常轻松。 现在谈谈我的转行经历，学习和几次面试的总结。 1、既然现在的工作做的不快乐，而且又有了转行的想法，那当然就果断点付出行动。由于当时没有条件脱产学习，所以就选择了麦子学院的课程。 2、在学习的过程中并没有想象的容易，学习的课程会起到快速入门的作用，而且会培养编程方面的思路。但是同学们也要自己多做练习，拓展，查阅资料。就算是天才也不会只看一遍视频就掌握所有的只是，所以在这里引用一句老师常说的话“多听，多写，多看”，这样才能更好的掌握技术。 3、最好不要学习完视频就急于找工作，因为已经不是应届生了，公司也不会给你很多的时间从实习开始做起。所以在学习完课程以后至少要做出一个自己的项目，最好是两个或两个以上，这样找工作才有说服力。在这里说一下，我们每节课程视频结束的项目作业和学院的毕设项目，就可以当做很好的项目经验。 4、基础学习很重要，大家一定要多多练习。大多数公司面试的时候都会有笔试题，但是放心，只要基础够硬，都是很容易的题。也可以上网查写笔试题自己联系。 5、公司的招聘信息都会有“任职要求”，会出现大家没听过或者没用的技能。这个也不要怕，在接到面试通知以后，可以跟自己的专业老师交流一下，请教一下老师这些技能的常用方法和特点。在这里还要感谢一下李希老师，在这方面给了我很大的帮助。 6、面试的时候不要紧张，要想清楚问题再回答。面试并不需要做到对答如流，但一定要思路清晰，这点可以在面试之前请老师做一次模拟面试。（还要说一点我面试中遇到的，当被问到不会的问题时，如果大家真的没听过，一定要如实承认，不要不懂装懂，因为在专业人士面前非常容易被识破） 7、还有一点与大家共勉，面试失败以后不要灰心，总结每次面试的不足，下一次一定会做的更好，我也是失败四五次以后才找到工作的。 最后说一说工作后的感想，由于刚刚入职，看公司代码非常吃力，压力很大。所以顶住开始的压力非常重要，遇到不懂得一定要主动多问同事和老师，因为不论是你的领导还是同事，都希望有一个积极，努力的工作伙伴。 希望大家都找到理想的工作岗位，一起加油吧。
